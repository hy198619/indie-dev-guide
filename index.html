📋 第一部分：你必须搞定的备案与许可

假设你已经注册了个体工商户或一人有限责任公司，经营范围包含互联网/电商相关。以下是面向 C 端付费产品的合规清单，按优先级排列：

1. ICP 备案（必须 ✅）

法规依据：《非经营性互联网信息服务备案管理办法》（2023年修订版）、工信部令第33号

核心要求：只要你的网站/网页服务器在中国大陆境内（包括火山引擎 ECS、阿里云、腾讯云等），就必须 ICP 备案。否则无法绑定域名，国内 CDN 和云厂商会阻断 80/443 端口。

主体类型备案名称所需材料审核周期备注
个人 个人姓名 身份证正反面、手机号、人脸核验 7-20 工作日 个人备案不得含商业/经营性内容
个体户 营业执照名称 营业执照、法人身份证、域名证书 7-20 工作日 可做简单经营性网站
有限公司 公司名称 营业执照、法人身份证、域名证书 7-20 工作日 可做完整商业网站，推荐 ✅

建议：用你的公司主体做 ICP 备案。个人备案不允许做付费产品，公司备案最稳妥。

2. 小程序备案（如果用微信小程序/抖音小程序，必须 ✅）

法规依据：工信部《关于开展移动互联网应用程序备案工作的通知》（2023年8月发布，2024年4月起强制执行）

核心变化：2024年4月起，所有小程序（微信、抖音、支付宝等）必须完成备案才能上架。未备案的小程序将被下架。

关键规则：

  • 小程序备案不独立，必须先完成域名 ICP 备案
  • 小程序备案主体必须与 ICP 备案主体一致
  • 一个公司主体可备案多个小程序

3. ICP 许可证 / EDI 许可证（视情况 ⚠️）

许可证全称适用场景你需要吗？
ICP 经营许可证 增值电信业务经营许可证（仅限互联网信息服务） 平台型网站，提供信息发布、会员服务等（如 58 同城、知乎） 暂不需要
你卖自己的产品/数字商品，属于自营，不是平台
EDI 许可证 在线数据处理与交易处理业务 电商平台、第三方交易处理（如淘宝、京东） 暂不需要
你自营商品/服务，不涉及第三方商家入驻
网络文化经营许可证 文网文 游戏、音乐、动漫、网络表演等 不需要
数字商品/工具类通常不属于此范畴

结论：现阶段你只需要 ICP 备案（公司主体），不需要办理任何经营许可证。当你的产品做大、涉及平台模式或第三方交易时再考虑。

关于营利性的常见疑问：ICP 备案名称虽为"非经营性"，但实际覆盖绝大多数自营商业网站。卖自己的数字商品、工具、课程等，都属于"利用自有平台销售自有商品"，只需 ICP 备案 + 营业执照即可。只有当你做"平台"让别人入驻、或提供有偿信息服务（如付费会员制内容平台）时，才需要 ICP 经营许可证。

4. 公安联网备案（必须 ✅）

ICP 备案通过后 30 天内，需在全国互联网安全管理服务平台完成公安备案。流程简单，纯线上操作，1-3 个工作日。与 ICP 备案并行即可。

5. 其他（暂不需要，了解即可）

  • ⏭️ APP 备案：如果你做原生 APP（iOS/Android），2024年起也需要备案。但现阶段你不需要。
  • ⏭️ 数据出境安全评估：如果你的服务器和数据都在国内，不涉及。
  • ⏭️ 个人信息保护影响评估（PIA）：如果你的产品收集敏感个人信息（生物特征、行踪轨迹等），需要。一般工具类产品不需要。

📦 第二部分：你现在需要准备的清单

已有 ✅

  • 个体工商户或公司营业执照
  • 经营范围含互联网/电商相关

需要补充 📋

  • 域名：购买一个域名（.com/.cn，约 30-60 元/年），阿里云/腾讯云均可
  • ICP 备案：以公司名义提交备案（火山引擎/阿里云/腾讯云后台均可操作）
  • 公安联网备案：ICP 备案通过后 30 天内完成
  • 微信商户号：如需微信小程序/公众号支付，需以公司名义申请微信支付商户号
  • 对公银行账户：有限公司需开对公账户，用于收款和税务
  • HTTPS 证书：云厂商免费 SSL 证书或 Let's Encrypt（免费）

🛤️ 第三部分：四种产品形态路径对比

📱 微信小程序

微信生态内 需备案
优势
• 微信支付原生集成，用户付费路径最短
• 微信内免域名备案即可分享（但上架需备案）
• 获客/裂变能力最强（群/朋友圈/公众号）
• 开发框架成熟，云开发可免服务器
劣势
• 审核严格：特定行业/内容可能被拒
• 必须 ICP 备案 + 小程序备案（2024年起强制）
• 需申请微信商户号（公司主体，约 1-2 周）
• 每年 300 元认证费（企业主体）
成本估算
域名备案0 元（云厂商免费代办）
小程序认证300 元/年
微信商户费率0.6%（数字商品类）
服务器可用微信云开发（免费额度 5 万读/天）

🤖 AI 平台应用（扣子 Coze / 扣子空间）

AI 平台 零代码
优势
• 完全免备案、免服务器、免部署
• 开发极快：拖拽 + Prompt 即可上线
• 可发布到豆包、飞书、微信等平台
• 适合 AI 对话类、咨询类产品
劣势
• 支付能力弱：平台本身不支持 C 端直接付费
• 定制化程度低：UI 受平台模板限制
• 数据所有权模糊：用户数据在平台方
• 不适合"展示+下载"模式
• 依赖平台稳定性，存在被封风险
成本估算
平台费用免费（基础额度）
高级功能按 Token 计费，用量大时成本上升
备案不需要
支付需自行对接第三方，复杂

🌐 H5 Web App

最灵活 需备案(国内)
优势
• 完全自主：UI、功能、数据全控
• 一套代码多端运行（手机/PC/微信内均可）
• 可用 Vercel/Cloudflare 海外部署（免备案测试）
• 支付灵活：微信/支付宝/第三方皆可接
• 代码可复用到小程序（大部分逻辑通用）
劣势
• 国内部署必须 ICP 备案
• 微信内打开海外域名可能被拦截
• 支付接口需自行对接（微信 JSAPI / 支付宝 H5）
• 需自行处理 SEO、PWA 等优化
成本估算
域名30-60 元/年
服务器(ECS)火山引擎 ECS（或已有服务器，0 元新增）
Vercel 海外版免费（Hobby 计划）
SSL 证书免费（Let's Encrypt / 云厂商）
支付费率微信 0.6% / 支付宝 0.6%

📲 原生 APP

成本高 需备案
优势
• 体验最好：原生性能、推送通知
• App Store 内购支付（iOS），用户体验好
• 品牌感最强，适合长期运营
劣势
• APP 备案强制（2024年起），流程比网站更复杂
• App Store 审核严格，虚拟商品抽成 30%
• 安卓国内需上架多家商店（华为/小米/OPPO/vivo）
• 开发成本最高（需适配 iOS + Android）
• 用户下载门槛高（不如扫码打开网页）
成本估算
Apple 开发者账号99 美元/年（约 700 元）
安卓上架各平台免费注册，但需软著
软件著作权300-2000 元（加急）
APP 备案与网站备案类似，0 元
苹果抽成30%（虚拟商品）

🎯 第四部分：综合推荐路径

⭐ 最佳路径：H5 Web App → 微信小程序（两阶段）

基于你的情况（有公司、有 ECS、AI Coding 开发、C 端付费、初期销量预期不高），推荐分两阶段走：

阶段一：H5 Web App（1-2 周上线，验证 PMF）

技术架构

  • 前端：纯 HTML/CSS/JS（或 Vue/React 单页应用），部署在 Vercel（海外测试）或云服务器（备案后）
  • 后端：Python FastAPI 跑在云服务器上，封装核心业务 API
  • 支付：先用面包多 / 爱发电 / 小报童等第三方数字商品平台嵌入，零开发成本
  • 域名：购买域名并启动 ICP 备案（7-20 天），期间用海外 Vercel 测试

为什么先做 H5？

  • 备案期间可以先在 Vercel 上跑通全流程
  • 微信内虽然可能偶尔被拦截，但可以通过"浏览器打开"引导绕过
  • 支付用第三方平台，无需申请商户号，今天就能收钱
  • 代码写完后，备案一下来，迁移到 ECS + 绑定域名，一天搞定

阶段二：微信小程序（H5 验证后，1-2 周迁移）

触发条件：H5 日活稳定 > 50 人，或有明确付费转化

  • H5 代码大部分可直接复用到微信小程序（Vue/React 适配成本很低）
  • 后端 API 完全复用，只需换域名指向
  • 申请微信商户号（已有公司资质，约 1-2 周）
  • 提交小程序备案（基于已有 ICP 备案，约 3-7 天）
  • 注意：应用名称和描述避开敏感词，使用合规表述

⚠️ 第六部分：独立开发者避坑指南

1. 不要在微信里发未备案域名链接

微信对未备案域名（特别是海外托管如 vercel.app、pages.dev）拦截率极高。测试阶段用短链工具或引导用户复制链接到浏览器打开。

2. 特定行业产品的"合规包装"

无论是小程序还是网站，避免使用可能被审核判定为敏感的词汇。建议：

  • 提前研究同类上架产品，参考其命名和描述方式
  • 用更中性、专业的表述替代可能引起审核关注的词汇
  • 如不确定，可以先提交测试版本看审核反馈

3. 一人有限责任公司的税务注意

  • 一人有限公司需独立记账报税，不能和个人账户混用
  • 年收入 10 万以下可享受小微企业税收优惠
  • 数字商品收入按"技术服务费"或"软件销售收入"申报
  • 建议找代理记账公司（约 200-300 元/月）

4. 支付接口选择优先级

  1. MVP 阶段：面包多 / 爱发电（零开发，嵌入即用，抽成 5-8%）
  2. 过渡阶段：微信 H5 支付 / 支付宝当面付（需商户号，费率 0.6%）
  3. 成熟阶段：微信小程序原生支付（体验最好，需商户号 + 小程序）

5. 数据合规底线

  • 收集用户信息，需在页面添加隐私政策链接
  • 不需要用户注册登录也能完成购买（降低门槛）
  • 不要收集手机号、身份证号等敏感信息
  • 用户数据本地处理，不要明文存储

🔀 第八部分：一句话决策

已有公司 + 已有服务器 + 要 C 端付费 + 初期量小
→ 先做 H5 Web App（Vercel 测试 → 备案后切 ECS）
→ 跑通后迁移到微信小程序
→ 原生 APP 暂不考虑
→ AI 平台应用仅适合对话类产品，不适合数字商品售卖

本报告基于 2024-2025 年工信部、微信开放平台、阿里云等官方公开信息整理

法规可能更新，建议在关键节点（备案、上架）前复核最新政策
